# Catálogo de Videojuegos

## Descripción

Aplicación web desarrollada con HTML, CSS y JavaScript que permite explorar un catálogo de videojuegos de forma interactiva.

La información de cada videojuego se obtiene desde un archivo JSON utilizando la Fetch API y se muestra dinámicamente en la interfaz.

Este proyecto fue realizado como práctica del ciclo formativo de Desarrollo de Aplicaciones Multiplataforma (DAM).

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- JSON
- Fetch API

---

## Funcionalidades

- Visualización de un catálogo de videojuegos mediante tarjetas.
- Consulta de información detallada de cada videojuego.
- Búsqueda por nombre.
- Filtrado por género.
- Filtrado por letra inicial.
- Actualización dinámica del contenido sin recargar la página.

---

## Origen de los datos

La información de los videojuegos se almacena en un archivo JSON incluido en el proyecto y se carga dinámicamente mediante la Fetch API.

---

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el proyecto con un servidor local (por ejemplo, Live Server en Visual Studio Code).
3. Acceder a la página desde el navegador.

> **Nota:** Debido al uso de `fetch()`, algunos navegadores pueden bloquear la carga del archivo JSON si el proyecto se abre directamente con `index.html`. Por ello se recomienda utilizar un servidor local.
